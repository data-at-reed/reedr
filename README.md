<!-- badges: start -->
<!-- badges: end -->

The goal of reedr is to provide datasets for Reed College.

## Installation

You can install the development version of reedr from [GitHub](https://github.com/) with:
``` r
# install.packages("pak")
pak::pak("data-at-reed/reedr")
```

## Example

Load graduation rate data:
``` r
library(reedr)
data(grad_rates)
data(grad_rates_2011_2024)
```
